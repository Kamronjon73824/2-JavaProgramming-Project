# Project 2 - Spring Boot REST API

## Description
This project shows a REST API for managing products using Spring Boot, Spring Data JPA, and H2 database.

## API Testing
The application was tested using Swagger UI.
* **POST /api/products**: Successfully created a product.
* **GET /api/products**: Successfully retrieved the list of products.
* **H2 Console**: Verified data persistence in the database table.
